🚀 글로벌 프로젝트 테스트

Cloudflare Pages + GitHub 자동 배포 테스트용 메인 페이지입니다.

📌 프로젝트 개요

이 페이지는 배포 파이프라인(Git -> GitHub -> Cloudflare Pages)이 정상적으로 작동하는지 확인하기 위한 테스트 파일입니다.

아래와 같은 기능들을 순차적으로 연동할 예정입니다:

  • AI 모델: Teachable Machine (동물상 테스트 등)
  • 수익화: Google AdSense, Stripe/Polar 글로벌 결제
  • 고객 소통: Disqus(댓글), Formspree(문의폼)

🛠️ 기능 테스트 영역

나중에 API나 백엔드 기능이 추가될 자리입니다.

✉️ 문의하기 (Formspree 예정)

Formspree와 연동하여 백엔드 없이 이메일을 받을 수 있는 폼입니다.
